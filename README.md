# Dictionary App
This is a sample dictionary app developed using Jetpack compose based on MVVM clean architecture.

**Dependencies**

- Jetpack Compose
- Hilt
- Room(Offline caching)
- Retrofit

**API**

https://dictionaryapi.dev/
